# MySyukatsu  

就活を頑張る大学生に向けた就職活動管理アプリ  
  
## 概要  

自分の周りに就活生が多く、「就活の状況管理が大変」という声があったので、就活状況管理に特化したアプリを作成しました。  
  
## 使用言語・環境  

言語・フレームワーク：　Ruby(2.6.3) , Ruby on Rails(6.0.3)  
環境：cloud9  

## 機能  

・ログイン認証  
・ユーザーごとの企業データ管理  
・管理者によるユーザー管理(ユーザーのデータ閲覧、ユーザー削除機能は今回なし)
